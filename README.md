# Fluxify React (Vite) - Component-based

## How to run

1. Install dependencies:
   ```
   cd fluxify-react
   npm install
   ```

2. Run dev server:
   ```
   npm run dev
   ```

3. Open the URL printed by Vite (usually http://localhost:5173)
